# Sistema HealthCore

Sistema de gerenciamento hospitalar inteligente para organização do atendimento de pacientes.

---

## Descrição

O **HealthCore** é um sistema desenvolvido para auxiliar no gerenciamento do fluxo de atendimento em ambientes hospitalares.

Seu objetivo é tornar o processo de atendimento **mais justo, organizado e eficiente**, utilizando critérios de prioridade médica. O sistema considera:

- nível de gravidade do paciente;
- condições preferenciais;
- tempo de espera;
- ordem de chegada.

O sistema permite:

- cadastro de pacientes;
- organização automática da fila de atendimento;
- chamada do próximo paciente;
- acompanhamento das ações realizadas no sistema.
