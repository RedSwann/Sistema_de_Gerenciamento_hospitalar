# Sistema HealthCore

Sistema de gerenciamento hospitalar inteligente.

## Descrição
O projeto consiste em desenvolver um sistema para organização do atendimento de pacientes em um ambiente hospitalar. Tem como objetivo garantir um atendimento mais justo e eficiente, priorizando pacientes com maior gravidade, considerando também condições preferenciais e o tempo de espera, mas respeitando a ordem de chegada dos pacientes.
